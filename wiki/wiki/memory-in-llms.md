# Memory in LLMs

Memory refers to the ability of an LLM to retain, recall, and interpret information across conversation turns spanning days, months, or years.

## Core Problem
The challenge of creating cumulative, evolving conversational recall without losing accuracy or falling into the "raw vs. derived" trap.
