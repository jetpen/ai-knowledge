# Agent Network Protocol (ANP)

## Overview
The Agent Network Protocol (ANP) is designed as a foundational network layer (the "HTTP of the Agentic Web") for global agent collaboration, focusing on interconnectivity, native API interaction, and efficient agent negotiation.

## Protocol Architecture
1. **Identity Layer**: Uses Decentralized Identifiers (W3C DID standards) for authentication and end-to-end encrypted communication.
2. **Meta-Protocol Layer**: Enables dynamic, automatic organizational structures within agentic networks.
3. **Application Layer**: Provides semantic capability descriptions for efficient protocol management.

## Vision
To build an open, secure, and cost-effective collaboration network capable of supporting billions of autonomous AI agents.
