# [[agentic-protocols]]: [[anp]] vs [[mcp]] vs [[a2a]] vs [[ag-ui]]

## Overview
The AI landscape is rapidly developing diverse protocols to standardize [[agent-communication]].

| Protocol | Primary Focus |
|----------|---------------|
| **ANP**  | Broad agentic collaboration network (HTTP of agentic web) |
| **AG-UI**| Agent ↔ User interaction & UI |
| **MCP**  | AI ↔ External tools and data |
| **A2A**  | Agent ↔ Agent messaging & interoperability |

## Observations
- **ANP** focuses on a massive, decentralized scale using [[w3c-did]] for identity.
- **AG-UI**, **MCP**, and **A2A** are focused on specialized layers of the agent stack (User, Tools, Coordinator).
