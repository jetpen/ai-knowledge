# [[agentic-protocols]]: [[ag-ui]] vs [[mcp]] vs [[a2a]]

## Overview
The AI ai-[[ai-ecosystem]] has developed three primary open standards to manage agentic interactions:

1. **AG-UI (Agent–User Interaction Protocol):** Connects AI agents to user-facing applications (Agent ↔ User).
2. **MCP ([[model-context-protocol]]):** Connects agents to external tools and data sources (Agent ↔ External Resources).
3. **A2A (Agent to Agent):** Connects agents to other agents (Agent ↔ Agent).

## Comparison Matrix

| Aspect | AG-UI | MCP | A2A |
|--------|-------|-----|-----|
| **Core Focus** | Agent ↔ User Interaction | Agent ↔ Data/Tools | Agent ↔ Agent Interaction |
| **Primary Goal** | Real-time, interactive UI/UX | Data & Tool accessibility | [[agent-interoperability]] & workflow |
| **[[linux-foundation]]** | Yes | Yes | Yes |
