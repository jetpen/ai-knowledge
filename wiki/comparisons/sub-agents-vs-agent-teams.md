# Sub-Agents vs Agent Teams

| Dimension       | Sub-Agents                  | Agent Teams                  |
|-----------------|-----------------------------|------------------------------|
| **Coordination**| Parent-controlled, isolated | Lead-led, shared context     |
| **State**       | Stateless, one-shot         | Persistent, interactive      |
| **Communication**| None between sub-agents     | Real-time via shared layer   |
| **Focus**       | Execution & compression     | Collaboration & adaptation   |
| **Scalability** | Parallel subtasks           | Interdependent workflows     |

## Synthesis
Choose sub-agents for clean delegation and parallelism. Use agent teams for tasks requiring ongoing coordination.
