# A First-Principles Guide to Agent Memory
**Author**: Akshay (@akshay_pachaure)
**URL**: https://x.com/i/status/2043745099792953508
**Ingested**: 2026-04-13

## Summary
The article explores the limitations of current LLM context windows (leading to "amnesia" and "lost in the middle" phenomena) and proposes a layered memory architecture for AI agents.

## Key Cognitive Framework
- **Sensory Memory**: Embedding representations.
- **Working Memory**: LLM context window.
- **Long-term Memory**: Externalized databases (Vector, Relational, Graph).

## Memory Maturity Layers
1. **Layer 1 (Python List)**: Volatile, unbounded growth.
2. **Layer 2 (Markdown files)**: Persistent, lacks semantic scale.
3. **Layer 3 (Vector Search)**: Semantic similarity, fails at relational reasoning.
4. **Layer 4 (Unified Solution)**: Cognee framework (combining Vector, Relational, Graph).

## Key Terms
- **Cognee**: Open-source unified memory framework.
- **Memory Types**: Episodic (events), Semantic (facts), Procedural (skills).
- **Architecture**: Retrieval-Augmented Generation (RAG) + Knowledge Graphs.
