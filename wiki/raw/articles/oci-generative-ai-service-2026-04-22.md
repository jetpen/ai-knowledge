# OCI Generative AI Service

OCI Generative AI is Oracle Cloud Infrastructure's \"Enterprise AI\" Platform for building, deploying, and governing AI applications at scale. Features frontier models, built-in agent development tools, vector stores, connectors, managed context retention and memory, hosted runtime for agent development and orchestration. Enterprise controls: IAM, guardrails, observability, auditability.
