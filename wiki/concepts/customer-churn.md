# Customer Churn

**Customer Churn** is the rate at which customers stop using a product/service. High churn kills growth; predict/prevent via ML (cohort analysis, usage signals).

## Agentic Context
Agents monitor engagement, trigger retention workflows (personalized outreach, feature nudges).

## Metrics
Churn Rate = (Lost Customers / Total) * 100
