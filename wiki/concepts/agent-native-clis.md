Non-interactive by default.\n2. Structured output (JSON).\n3. Actionable errors.\n4. Safe retries (idempotency).\n5. Bounded responses.\n\n### Tier 2: Compounding\n6. Vocabulary consistency (--yes, --force).\n7. Introspection (--help, schema, manifest).\n8. Async execution (--wait).\n9. Profiles for identity.\n10. Two-way I/O (artifacts, feedback).\n\n**Examples:** Cloudflare Wrangler (TS schema → CLI/SDK/MCP/OpenAPI; ~3k ops <1k tokens), HeyGen CLI (video gen/polling).\n\nRelates: [[agent-harness]], [[mcp]], [[cloudflare-cli]]\n\nSee: [[trevin-chow]]
