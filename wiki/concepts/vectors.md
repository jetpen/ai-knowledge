# Vectors

Fundamental data structure in linear algebra and ML: 1D array of numbers, e.g., `[2, 4, 7]`.

## Operations
- Addition: element-wise
- Scalar multiplication
- Dot product: `sum(a_i * b_i)` – cosine similarity, projections
- Norm: `sqrt(sum(x_i^2))` – length/magnitude

## ML Usage
- Embeddings (word2vec, token embeds)
- Hidden states in RNNs/Transformers
- Gradients in backprop
