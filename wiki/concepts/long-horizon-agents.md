# Long-Horizon Agents

Agents sustaining **roles** (ongoing responsibilities) across runs, not just bounded **tasks**.

Require [[environment-engineering]] beyond [[agent-harness]] for coherence, learning, improvement.

- Roles need durable invariants: memory evolution, evals, reviews.
- Test: [[harness-swap-test]] – what survives harness change?

From [[li-jeffrey]] / [[hola-os]]: Environment sustains roles; harness executes tasks.
