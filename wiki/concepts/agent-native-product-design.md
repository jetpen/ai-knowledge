# Agent-Native Product Design

Shift in 2026: Designers split → pixel-pushers (Figma-heavy) vs agent-native (80% planning/20% execution via agents).

Per [[rico]]: Agents handle production; designer strategizes/dispatches/reviews/ships.

## 4 Core Artifacts
1. **strategy.md**: Problem/approach/user/metrics/tracks (Rumelt/GSS-inspired).
2. **design.md**: Tokens (color/space/typo), constraints (e.g. "no blue buttons").
3. **CLAUDE.md/AGENTS.md**: Tech stack, codebase rules, agent prompts.
4. **product-pulse**: Analytics routine (e.g. weekly retention agent-review).

## New Loop (5h vs 5d)
- Strategy file → Brief (spec) → Dispatch (Cursor/Claude) → PR review → Ship URL.
