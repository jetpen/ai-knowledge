# Matrices

2D array (rows x columns) of numbers. Core to NN weights and linear transformations.

## Operations
- Transpose: rows ↔ columns
- Multiplication: `C = A @ B` – forward pass
- Determinant, inverse (square matrices)

## ML Usage
- Weight matrices in dense/conv layers
- Attention matrices (QK^T)
