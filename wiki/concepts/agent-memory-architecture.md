# Agent Memory Architecture

Current research categorizes AI memory into two primary paradigms based on architectural focus.

## 1. Memory Backends (Fact Recall)
Tools focused on "what the AI remembers."
* [[mem0]]
* [[mempalace]]
* [[supermemory]]


## 2. Context Substrates (Continuous Reasoning)
Architectures focused on "what context the AI works inside."
* **Mechanism**: Agent reads/writes to human-readable structured files (Markdown).
* **Workflow**: Continuous modification of the context itself (no hidden state).
* **Usage**: Multi-session/multi-project continuity, reasoning over evolving work.
* **Examples**: [[openclaw]].
