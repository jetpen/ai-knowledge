# codejunkie99/brain

Git-backed long-term memory for AI coding agents.

## Overview
Rust-based portable memory layer for AI agents. Designed to persist across sessions, models, and tools (e.g., Claude Code, Cursor). Provides a protocol for harnesses to share a common "brain".
