# SMFS

Supermemory Filesystem: Mountable FS for agents combining semantic (RAG-like) search with POSIX ops (grep/ls/cat).

## Features
- Semantic grep: vector DB query → file paths/line nums.
- Scoped to cwd.
- Handles PDFs/transcripts where plain grep fails.
