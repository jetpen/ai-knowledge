# Agentic Protocols Comparison

## Overview/Sources
Primary agentic protocols: AG-UI (Agent-User Interaction), MCP (Model Context Protocol), A2A (Agent2Agent). Sources from Linux Foundation-backed standards enabling AI ecosystem interoperability.

## Description
In agentic AI systems, protocols standardize communication for scalable, interoperable workflows. AG-UI bridges agents to UIs for real-time interaction, enabling seamless human-AI collaboration in apps like copilots. MCP connects agents to tools/data sources, providing secure context injection for enhanced reasoning without proprietary lock-in. A2A facilitates peer-to-peer agent coordination, supporting multi-agent orchestration, task delegation, and collective intelligence in long-horizon tasks. Comparison: AG-UI focuses user-facing sync; MCP tool/resource access; A2A agent swarms. Together, they form the TCP/IP for agents, reducing framework silos (LangGraph, CrewAI, Deep Agents) and accelerating enterprise adoption via open standards. (112 words)
