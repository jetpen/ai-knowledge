# Module

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
## Agentic Application

**Module** groups related agent functionalities for cohesion.

**Forces**:
- Cognitive load.
- Deployment units.
- Team ownership.

**Solution**:
1. High cohesion, low coupling.
2. Namespace isolation.
3. Plugin loading.

**Examples**:
- **Planning Module**: Decomp + select.
- **Memory Module**: Short/long-term.
- **Execution Module**: Tools + act.
- **Eval Module**: Metrics/logs.

Scales agent complexity.
