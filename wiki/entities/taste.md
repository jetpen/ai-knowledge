## Agentic Application

Taste pattern embeds subjective preferences into agent decision-making, guiding choices beyond objective metrics—like code elegance, creativity, or risk appetite. Agents maintain taste profiles as state, scoring options.

Agent example: Code gen agent rates implementations on "taste": Pythonic > verbose, favoring functional over imperative per team style [[ubiquitous-language]]. Content agent selects edgy topics for viral potential.

Pairs with [[tradeoff]] for balanced intent. Enables personalized agents, from conservative finance bots to innovative product ideation. Evolves via feedback loops, aligning with [[value-proposition]]. (92 words)
