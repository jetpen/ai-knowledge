# Contract

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
Formal agreement between [[consumer]] and provider defining inputs/outputs, errors, SLAs (e.g., [[schema-serialization]], OpenAPI, gRPC). Enables [[dependency]] inversion.

## When to Use
- Inter-agent communication
- Tool interfaces
- Microservices
