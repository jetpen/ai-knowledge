## Agentic Application
Problem pattern frames the core user pain agents solve, guiding scope. Articulate as 'Users struggle with X because Y, wasting Z time/cost.' For agentic: 'Manual code review takes hours; agent reviews PRs in minutes with 95% accuracy.' Validate via customer interviews, quantify impact. Agents use problem statement in system prompts for focus. Evolves with [[product-market-fit]]. Pitfalls: solving non-problems; test with MVPs. Foundation for [[value-proposition]]. (128 words)
