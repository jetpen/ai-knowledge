# AI Accelerator Packs

## Overview/Sources
OCI AI Accelerator Packs: pre-built, one-click AI solution stacks for rapid deployment of agentic workloads on Oracle Cloud Infrastructure.

## Description
AI Accelerator Packs from Oracle Cloud Infrastructure (OCI) package full-stack agentic applications—models, vector DBs, orchestration, monitoring—into deployable blueprints. In agentic AI, they enable instant spin-up of production agents for RAG, search chatbots, or multi-agent systems, slashing setup from months to hours. Key packs: GenAI Search Chat Agent, Document Intelligence. Supports frameworks like LangChain, integrates Bedrock-like services, with auto-scaling, security, and cost optimization. Agentic context: Facilitates enterprise-scale long-running agents with episodic memory persistence, tool access via MCP, and UI via AG-UI. Bridges cloud infra to AI-native ops, competing with AWS SageMaker, GCP Vertex. (124 words)
