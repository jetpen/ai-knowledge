## Agentic Application

User pattern treats end-users as core entities with personas, histories, and contexts—driving personalization in agentic apps.

Agent example: Support agent retrieves user state [[state]], tailors advice per profile (power vs novice). Marketing agent segments for targeted [[value-proposition]].

Centers product around humans, informs [[user-story]]. Evolves with feedback. From chatbots to enterprise CRMs, anchors agent intelligence to real needs. (91 words)
