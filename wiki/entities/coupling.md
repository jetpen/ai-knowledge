# Coupling

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
Degree of interdependence between modules/agents. **Tight coupling**: shared state/knowledge (fragile). **Loose coupling**: async, contracts, events (scalable).

Types: content, common-impact, data.

## When to Use
Minimize via [[abstraction]], [[interface]].
