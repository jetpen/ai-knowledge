     6|# Agent Network Protocol (ANP)
     7|
     8|The **Agent Network Protocol (ANP)** aims to be the "HTTP of the Agentic Web," enabling seamless communication and collaboration between diverse AI agents.
     9|
    10|## Overview
    11|ANP provides a decentralized framework designed to break down data silos and allow agents to discover, negotiate, and exchange information efficiently.
    12|
    13|## Core Pillars
    14|- **🌐 Interconnectivity**: Enables universal communication between agents.
    15|- **🖥️ Native Interfaces**: Focuses on API and protocol-based interaction.
    16|- **🤝 Efficient Collaboration**: Facilitates automated organization and negotiation.
    17|
    18|## Protocol Layers
    19|- **Identity Layer**: W3C DID-based decentralized auth.
    20|- **Meta-Protocol Layer**: Dynamic protocol negotiation.
    21|- **Application Layer**: Semantic-based capability description.
