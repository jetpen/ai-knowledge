# Agent Network Protocol (ANP)

The **Agent Network Protocol (ANP)** aims to be the "HTTP of the Agentic Web," enabling seamless communication and collaboration between diverse AI agents.

## Overview
ANP provides a decentralized framework designed to break down data silos and allow agents to discover, negotiate, and exchange information efficiently.

## Core Pillars
- **🌐 Interconnectivity**: Enables universal communication between agents.
- **🖥️ Native Interfaces**: Focuses on API and protocol-based interaction.
- **🤝 Efficient Collaboration**: Facilitates automated organization and negotiation.

## Protocol Layers
- **Identity Layer**: W3C DID-based decentralized auth.
- **Meta-Protocol Layer**: Dynamic protocol negotiation.
- **Application Layer**: Semantic-based capability description.
