# ADK 2.0

Agent Development Kit 2.0 by Google Cloud, launched at [[google-cloud-next-26]].

## Key Features
- **Graph-based workflows**: Directed graphs mixing deterministic nodes (business rules) and AI-driven nodes (LLM reasoning).
- **Collaborative agents**: Orchestration across multiple agents.
- **Formalized Skills framework**: Builds on prior agent skill patterns.
