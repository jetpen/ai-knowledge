# Ubiquitous Language

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
## Agentic Application

Ubiquitous Language pattern creates a shared, precise vocabulary binding agents, developers, and domain experts—crucial for agentic systems where LLMs interpret natural language.

Agent example: In a dev swarm, define "deploy" uniformly: git push + CI/CD trigger. Prompts mandate: "Respond using [[ubiquitous-language]] terms only." Code gen aligns with business via common lexicon.

Reduces errors from synonyms/ambiguity, evolves through workshops. Pairs with [[taste]] for nuance, [[user-story]] for scenarios. Scales team cognition to agent fleets. (96 words)
