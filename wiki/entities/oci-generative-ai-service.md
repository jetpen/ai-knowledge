# OCI Generative AI Service

Oracle Cloud Infrastructure's (OCI) **"Enterprise AI" Platform** for building, deploying, and governing AI applications at scale.

## Key Features
- **Frontier models** (pretrained + custom imports from Hugging Face/Object Storage)
- **Built-in agent development tools** with hosted runtime for orchestration
- **Vector stores**, connectors, managed context retention/memory
- **Enterprise controls**: IAM, [[guardrails]], observability, auditability
