# Judgment

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
## Agentic Application

**Judgment** pattern applies heuristics/taste in ambiguous decisions.

**Forces**:
- Incomplete data.
- Tradeoffs.
- Experience-based calls.

**Solution**:
1. Chain-of-thought + rubrics.
2. Multi-LLM voting.
3. Historical sims.

**Examples**:
- **Build/Buy**: Score options.
- **Arch Choice**: Risk assessment.
- **Prioritize Features**: Impact/effort.
- **Debug**: Root cause vote.

Sharpens agent wisdom.
