# LangSmith Deployment (LSD)

LangSmith Deployment (LSD) is the production runtime for running agents, paired with Agent Server for interfaces to assistants, threads, runs, memory, and scheduled jobs.

## Key Capabilities
- Durable execution via managed task queue and checkpointing.
- Supports [[deep-agents]] deploy.
