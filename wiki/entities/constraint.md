# Constraint

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
Explicit limits on agent behavior, resources, or scope (e.g., budget, token limits, safety rules, performance SLAs). Constraints guide decision-making, prevent overreach, and enable [[tradeoff]] analysis.

## When to Use
- Resource-bounded agents
- Compliance/safety ([[content-moderation]])
- [[spec-driven-development]]
