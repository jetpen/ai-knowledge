# WayFlow

WayFlow is an open-source Python library developed by Oracle designed for building sophisticated, AI-powered assistants. It emphasizes reusability, interoperability, and rapid development, serving as the reference runtime for the [Open Agent Spec](https://oracle.github.io/agent-spec/).

## Overview
WayFlow provides a flexible, modular architecture that supports both:
- **Agentic Architectures**: Autonomous assistants capable of sophisticated reasoning and task fulfillment.
- **Flow-based Architectures**: Fixed-structure workflows for more predictable, task-oriented interactions.

The library is designed to be agnostic, allowing seamless integration with various LLM providers, including OCI Gen AI, OpenAI, and Ollama.
