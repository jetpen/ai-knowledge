# Agent Network Protocol

ANP (Agent Network Protocol) is a network-layer specification for AI nodes to discover, connect, and route tasks in a distributed multi-agent architecture.

## Key Features
- Interoperability across diverse LLM systems.
- Robust state management for long-running workflows.
- Standardized tool integration pathways.
