# OCI Generative AI Agents

OCI Generative AI Agents provide a managed RAG (Retrieval-Augmented Generation) layer, best suited for building document-grounded knowledge assistants.

## Overview
As part of the OCI AI Services stack, these agents allow for rapid development of systems that retrieve context from private documents to provide accurate, grounded answers.
