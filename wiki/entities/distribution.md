# Distribution

**Agentic Coding Pattern** from [[agentic-coding-patterns]] (AiPatternBook REPL).

## Context
## Agentic Application

**Distribution** pattern from \"Product Judgment\" focuses agents on channels to reach users, scaling adoption beyond direct sales.

**Forces**:
- Viral growth needs low-friction sharing.
- Platform dependencies (App Stores, marketplaces).
- Freemium to paid conversion funnels.

**Solution**:
1. Map channels: OSS GitHub, HF Spaces, agent marketplaces (Dust.tt).
2. Agents A/B test landing pages, SEO via tools.
3. Embed shareability: API keys, one-click deploys.
4. Track metrics: DAU, virality coeff.
5. Survey successes: LangChain PyPI downloads.

**Examples**:
- **Agent Marketplace**: Publish to Composio, track installs.
- **Viral Loop**: Agents generate/share custom variants.
- **Partnerships**: Integrate with Zapier for no-code dist.
- **Edge Deploy**: Cloudflare Workers for global latency.

Multiplies reach 10x; agentic optimization hits hockey-stick growth.
