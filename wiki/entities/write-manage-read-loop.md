# Write-Manage-Read Loop

## Definition
Agent memory cycle:
- **Write**: New info (observations, reflections).
- **Manage**: Prune, compress, consolidate.
- **Read**: Retrieve for context.

Most neglect Manage → noise/contradiction.

## In Practice
- OpenClaw: Heuristic policies for storage/escalation.
