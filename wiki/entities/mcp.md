# Model Context Protocol

MCP (Model Context Protocol) is an open standard designed to enable universal connectivity between LLM apps and local/remote data sources, significantly reducing integration surface area for agent tools.

## Key Features
- Interoperability across diverse LLM systems.
- Robust state management for long-running workflows.
- Standardized tool integration pathways.
