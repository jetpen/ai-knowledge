## Agentic Application

User Story pattern frames requirements conversationally: "As a [user], I want [goal] so that [benefit]"—prioritizing value delivery.

Agent example: Product agent generates stories from features: "As dev, want auto-testing so faster iteration [[zero-to-one]]." Backlog grooming via swarms.

Agile for agents, links [[use-case]] details to vision. Drives iterative builds, measures success via acceptance criteria. (88 words)
